(so you can take it easy)

 

What is Granger?

 

It's a tool that helps breeding, or simply cataloguing horses (and should theoretically work for hell horses as well). This is a short list of what exactly can Granger do for you:

 

·       Add horses into program database (or update if already exist) by simply smiling at one, then examining it before 5 seconds pass.

·       Saves traits, gender, father and mother, who is caring for the horse, as well as tracks pregnancy, grooming and not-in-mood periods.

·       Values each horse according to how you set each trait individual value.

·       Organizes horses into multiple, named herds (lists), view can be easily sorted and each horse inspected for good or bad traits.

·       Advises most optimal breeding within a herd, taking into account all known factors (inbreeding, traits and their values and if the horse is currently available for breeding). There are options to further configure how breeding AI makes it's judgments.

·       Herds can be renamed, deleted or merged.

·       Horses can be manually added, edited, deleted and moved between herds.

·       Accidental overwriting horse data by other same-named horses is prevented (by validating gender, parents and traits before updating the horse).

·       Burying a horse will mark it as dead in the program.

 

 

Well, that's in a nutshell. But how to use it?

 

In order to get a horse into the program database, you need to do the following:

- smile at the horse

- before 5 seconds passes, examine this horse

After that, you should see a popup informing about adding a horse to the database, or updating data about existing horse.

 

 

Note that Granger in Wurm Assistant versions older than 2.0 will capture this data ONLY FOR MAIN CHARACTER, as set in program configuration.
